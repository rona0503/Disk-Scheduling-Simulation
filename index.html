▸ Knowledge Base
① SEEK TIME
Time for the r/w head to move to the correct track/cylinder. Dominant cost. Range: 1–20ms. Minimized by scheduling algorithms.
② ROTATIONAL LATENCY
Time waiting for the disk to rotate so the correct sector is under the head. Avg = ½ rotation. ~2–8ms on modern drives.
③ DATA TRANSFER TIME
Time to actually read/write the data once positioned. Proportional to data size and rotational speed. Usually the smallest component.
TOTAL ACCESS TIME =
Seek + Rotational Latency + Transfer
